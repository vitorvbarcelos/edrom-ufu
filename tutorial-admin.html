COMEÇANDO

1. Entrar no painel

Acesse /admin.html (ex.: seusite.com/admin.html). A página é oculta — não aparece no menu.

CampoValor
Usuárioedrom
Senhaedrom2026

Para trocar a senha, abra js/admin.js nas primeiras linhas e edite ADMIN_USER e ADMIN_PASS. O botão Sair desloga.

⚠ Como os dados são salvos. Tudo fica no navegador (localStorage), sem servidor. O que você edita só vale naquele computador/navegador. Para migrar, use Exportar / Importar backup. É por isso também que o ranking do jogo é por navegador, não global.

2. Barra superior (sempre visível)

BotãoO que faz
Exportar backupBaixa um .json com tudo: leads, inscrições, seletivo, equipe, ranking e conteúdo.
Importar backupRestaura de um .json exportado (substitui os dados atuais).
Ver siteAbre o site numa aba nova.

03

Aba Leads

Contatos coletados pelo chat (robô) do site.

  • Filtro por tipo: Patrocínio ou Membro.
  • Busca: por nome, e-mail, empresa, curso ou área.
  • Exportar CSV: baixa a lista para Excel / Google Sheets.
  • Excluir: remove um lead da lista.

Candidatos que se inscreveram no processo seletivo (cada um recebe um código EDROM-0001).

AçãoEfeito
EtapaMuda a etapa do candidato (Inscrição → Desafio → Entrevista → Resultado). O candidato vê no site, na aba "Acompanhar inscrição".
StatusEm análise / Aprovado / Reprovado / Aguardando.
ExcluirRemove a inscrição.
BuscaPor nome, código ou e-mail.
05

Aba Processo Seletivo

Controle se o seletivo aparece no site e se aceita inscrições. Um selo mostra o status atual.

Controle rápido

BotãoEfeito no site
Abrir inscriçõesSeletivo ativo e aceitando inscrições (mostra "Inscreva-se").
Fechar inscriçõesContinua visível, mas sem inscrever ("INSCRIÇÕES FECHADAS").
Encerrar seletivoSome do site. Inscrições antigas continuam salvas.
Criar novoZera título, descrição e etapas para uma nova edição (começa fechado).

Editar seletivo: ajuste os checkboxes Ativo / Inscrições abertas, o Título, a Descrição e as Etapas (nome + data; "+ Adicionar etapa" e "×"). Clique Salvar seletivo.

✓ Estado atual: o seletivo já vem FECHADO. Quando a seleção começar, é só clicar Abrir inscrições.

06

Aba Equipe

Atualize o Conselho Diretor sem mexer no código. A seção "Equipe" do site lê exatamente daqui.

  • Adicionar membro: Nome, Cargo, Curso + foto por URL ou upload (o upload vira base64 e fica salvo).
  • Editar: "Editar" no card preenche o formulário; salve para atualizar.
  • Remover: tira o membro da equipe.

💡 Dica: para usar as fotos locais de backup, no campo "Foto por URL" coloque assets/equipe/pamella.png (e assim por diante).

07

Aba Ranking / Jogo

Moderação do Jogo do Robô.

  • Tabela do ranking: posição, nome, gols e data. Remover tira um score (útil se algum nome ofensivo passar).
  • Limpar ranking: zera tudo (pede confirmação).
  • Palavras proibidas: lista de bloqueio — nomes com essas palavras não salvam. Já vem com ~56 termos; "Adicionar" inclui, "×" remove.

O filtro entende disfarces com número/símbolo: 0→o, @→a, 3→e, 4→a. Então otári0 e c4ralho são bloqueados.

08

Aba Conteúdo

Textos e números do site.

  • Sobre e Destaque (a faixa "3º lugar CBR 2025").
  • Números do count-up: Anos, Membros, Competições, Premiações.
  • Patrocinadores: nome + URL do logo (é o que roda no marquee).

09

Backup — não perca os dados

1
Exportar backup → guarde o arquivo .json em local seguro.
2
Em outro PC/navegador → Importar backup restaura tudo.
3
Limpar o cache/dados do navegador apaga tudo — por isso o backup é essencial.
Lembre: as fotos da equipe e os logos dos patrocinadores vêm de URLs externas (Wix). Se algum dia saírem do ar, troque no admin pelas versões locais em assets/.
